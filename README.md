# CodeAlpha Credit Scoring Model

## Project Overview
This project was developed as part of the CodeAlpha Machine Learning Internship.

The objective of this project is to predict an individual's creditworthiness using machine learning techniques and financial data.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn

## Dataset
- German Credit Dataset

## Machine Learning Algorithm
- Random Forest Classifier

## Features Used
- Account Status
- Credit History
- Loan Duration
- Credit Amount
- Employment Status
- Savings Account
- Personal Status
- Age
- Housing
- Job
- Other Financial Attributes

## Project Workflow
1. Load the dataset
2. Preprocess the data
3. Encode categorical features
4. Split data into training and testing sets
5. Train the Random Forest Classifier
6. Predict credit risk
7. Evaluate model performance

## Results
- Credit Scoring Accuracy: **77%**

## Performance Metrics
- Accuracy Score
- Precision
- Recall
- F1-Score

## How to Run

1. Install required libraries:
   ```bash
   pip install pandas numpy scikit-learn
   ```

2. Run the program:
   ```bash
   python credit_scoring.py
   ```
